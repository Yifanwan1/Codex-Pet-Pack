# Codex Pet Pack

## 中文介绍

这里是我的 Codex 宠物合集仓库。每个宠物都有一个可以直接下载解压使用的标准宠物文件夹，标准文件夹里只包含：

- `pet.json`
- `spritesheet.webp`

宠物介绍、形象图、动作总览图和验证记录放在 `docs/` 目录里，避免污染可安装的宠物包。

## English Introduction

This is my Codex pet collection repository. Each pet has a directly usable standard pet folder. A standard pet folder contains only:

- `pet.json`
- `spritesheet.webp`

Descriptions, character images, contact sheets, and validation records live under `docs/`, so the installable pet package stays clean.
